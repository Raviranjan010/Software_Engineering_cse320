# Emerging Technologies

## Overview
Software engineering is continually evolving. Modern trends center around artificial intelligence, cloud computing, and faster delivery architectures to meet user demands at a global scale.

## 1. Cloud-Native Development
Applications built specifically to run in cloud environments (like AWS, Azure, Google Cloud).
- **Microservices**: Breaking applications into small, independent services rather than a monolith.
- **Containers (Docker)**: Packaging the app and all its dependencies into a single deployable unit.
- **Orchestration (Kubernetes)**: Managing thousands of containers automatically.

## 2. Low-Code and No-Code Platforms
These platforms allow developers and non-technical business users to build applications visually without writing extensive code.
- Reduces development time significantly.
- E.g., OutSystems, Appian, Bubble.

## 3. Serverless Architecture
Developers write code and functions without worrying about underlying server infrastructure.
- The cloud provider dynamically allocates machine resources.
- Pricing is based on the actual amount of resources consumed by an application, rather than on pre-purchased units of capacity.
- E.g., AWS Lambda, Google Cloud Functions.

## 4. AI in Software Development
Artificial intelligence is automating coding and debugging.
- **AI Coding Assistants**: Tools like GitHub Copilot and Amazon CodeWhisperer suggest whole blocks of code based on user comments.
- **Automated Code Reviews**: AI scanning code to identify security flaws and anti-patterns instantly.

**DEF**
**Machine Learning in SE**: Utilizing algorithms that allow software development tools to learn patterns from vast codebases (like GitHub) to assist human programmers.

## Practice Questions

**Q1. An architecture that breaks software applications down into small, loosely coupled, and independently deployable services is called:**
a) Monolithic Architecture
b) Serverless Architecture
c) Microservices Architecture
d) Client-Server Architecture
**Answer: c**
*Explanation: Microservices allow large applications to be built as a suite of modular components.*

**Q2. Which technology allows you to package an application and its dependencies together into an isolated environment?**
a) Git
b) Docker (Containers)
c) AWS Lambda
d) Selenium
**Answer: b**
*Explanation: Containers (like Docker) bundle an app and its dependencies so it can run predictably anywhere.*

**Q3. Tools like GitHub Copilot represent which emerging trend in Software Engineering?**
a) Low-Code Development
b) AI-assisted Software Development
c) Microservices
d) Cloud-native
**Answer: b**
*Explanation: Copilot utilizes LLMs (Artificial Intelligence) to assist in writing code.*
