## 📝 How to Answer Questions for Maximum Marks

### For 2-Mark Questions:
```
Structure:
- Definition (1 line)
- Example or key point (1 line)

Example:
Q: What is SRS?
A: Software Requirements Specification (SRS) is a comprehensive document 
that describes all external requirements for a software system. It serves 
as a contract between customer and developer (IEEE 830 standard).
```

### For 5-Mark Questions:
```
Structure:
- Definition/Introduction (2 lines)
- Main content with points (8-10 lines)
- Diagram (if applicable)
- Example (1-2 lines)

Example:
Q: Explain Waterfall Model
A: [Definition]
   [Diagram - always draw!]
   [Advantages in table]
   [Disadvantages in table]
   [When to use with example]
```

### For 10-Mark Questions:
```
Structure:
- Introduction (3-4 lines)
- Detailed explanation (10-12 lines)
- Diagram (mandatory, well-labeled)
- Comparison table (if applicable)
- Real-world example
- Advantages/Disadvantages
- Conclusion (2 lines)

Tip: Write 2-3 pages minimum for 10-mark questions
```

## 📊 Formulas to Memorize

### Testing:
```
Cyclomatic Complexity: V(G) = E - N + 2P
```

### COCOMO:
```
Effort (E) = a × (KLOC)^b
Time (D) = c × (E)^d
Staff Size = E / D
```

### Function Points:
```
FP = UFP × TCF
TCF = 0.65 + 0.01 × ΣFi
```

### PERT:
```
Expected Time (TE) = (O + 4M + P) / 6
Variance (V) = [(P - O) / 6]²
```

### CPM:
```
Slack = LS - ES = LF - EF
Critical Path = Longest path (zero slack)
```
