## Module 1: SDLC and Requirements Engineering

### Introduction to SE

**Q1. Software Crisis refers to:**  
a) Shortage of software developers  
b) Difficulty in writing correct software  
c) Problems in software development (cost overruns, delays, poor quality)  
d) Hardware failures  
**Answer: c**  
*Explanation: Software crisis refers to difficulties in software development leading to budget overruns, missed deadlines, and unreliable software.*

**Q2. Software Engineering is:**  
a) Just programming  
b) Systematic approach to software development  
c) Testing software  
d) Designing databases  
**Answer: b**
