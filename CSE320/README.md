# CSE320: SOFTWARE ENGINEERING - Complete Study Guide

## Course Information
- **Course Code**: CSE320
- **Course Title**: Software Engineering
- **Credits**: 3 (L: 3, T: 0, P: 0)
- **Objective**: Master software engineering principles, practices, and tools to develop high-quality software systems

---

## Course Outcomes (COs)

After completing this course, you will be able to:

| CO | Description |
|----|-------------|
| **CO1** | Explain the evolution of Software Engineering and apply SDLC models to analyze requirement-based problems |
| **CO2** | Apply software design principles to evaluate and analyze suitable system architectures for given scenarios |
| **CO3** | Use UML models to represent systems and analyze their correctness and consistency |
| **CO4** | Apply software testing techniques and analyze test results to determine software quality |
| **CO5** | Use project management and DevOps tools to analyze project planning, estimation, and process efficiency |
| **CO6** | Explain software quality standards and emerging technologies and analyze their impact on software development and maintenance |

## 🎯 How to Use These Notes for 100% Score

### Step-by-Step Study Strategy:

1. **First Pass (Understanding)**
   - Read each module sequentially
   - Focus on understanding concepts, not memorizing
   - Study all diagrams and examples

2. **Second Pass (Deep Dive)**
   - Re-read with focus on exam-oriented content
   - Practice all numerical problems (COCOMO, CPM, PERT, etc.)
   - Draw diagrams from memory

3. **Third Pass (Practice)**
   - Solve all MCQs and short answer questions
   - Time yourself while answering
   - Identify weak areas and revise

4. **Final Revision (Exam Prep)**
   - Use Quick_Revision.md for last-minute prep
   - Review key formulas and definitions
   - Practice drawing important diagrams

### Exam Tips:
- ✅ Always draw diagrams where applicable (even if not asked)
- ✅ Use real-world examples in answers
- ✅ Write advantages/disadvantages in tabular format
- ✅ Show step-by-step calculations for numerical problems
- ✅ Use proper terminology and definitions
- ✅ Attempt all questions (partial marks matter)
